What is shell?

Shell is a read-only shadow website from the GeniusWiki space. After link, space content will be automatically synchronized to shell.

Why link to a shell?

  • An easy way to set up a public website.
  • Different layout and style from GeniusWiki website.
  • In this scenario, GeniusWiki is a kind of content management system in the backend.
  • Optimized SEO - you have a Google hosted website.

How to link?

Choose a shell then input your Google account, GeniusWiki do following dirty jobs. Once it is done, you have a independent website hosted by Google. * Shell is intend to deploy to any Java or Python hosting. At present, we only support to deploy on Google App Engine. * To protect your privacy, we don't save your Google account information.
