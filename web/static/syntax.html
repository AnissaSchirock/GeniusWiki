Menu:

{menu}
and
{menuitem}
or
{menuitem:parent=pageTitle|title=menuTitle|order=10}

{menuitem} macro is invisible element in page. Put {menuitem} into the page that you want to link in menu. The parameters are all optional:
  • Parent is the parent page title. Note, it is NOT the parent menu title!
  • Title is replacement if page title is not so good as menu title.
  • Order is numeric that sorts the same level menu item in ascending sequence. Default is zero.
Macro {menu} will collect all {menuitem} in current space and display as a drop downn menu.
Menu can contain any cascade level. However, space must have at least one root level menu item, i.e., {menuitem} doesn't contain parent.
Please note, in default skin, the menu macro is already put into the top of page. Normally, you don't need to put a separated {menu} macro again.
